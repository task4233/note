      # Mujin 18 D うほょじご

      # 問題

      • D - うほょじご (opens new window)

      # 解法

      • (x,y)(x,y)(x,y)の組で操作を無限に続けられる ⟺\Longleftrightarrow⟺ いつか(x,y)(x,y)(x,y)の状態に戻ってくる
      • 制約が0≤x≤9990 \leq x \leq 9990≤x≤999であることに注目
      • 状態を全部追うことが可能っぽい
        • 一応, rev(x)rev(x)rev(x)の制約も同様に0≤rev(x)≤9990 \leq rev(x) \leq 9990≤rev(x)≤999なので
      • したがって全ての状態(0≤x≤9990 \leq x \leq 9990≤x≤999)で状態を回してみれば良い
